## 🛠 Tech Stack & Tools
At Long Gil Studio, we continuously optimize our development pipeline to deliver the best performance:
* **Engine:** Unity 6 (URP/HDRP Custom Pipelines)
* **Animation:** Motion Capture & Retargeting (Blender to Unity)
* **Localization:** Developing a multi-language system (VI, EN, CN, JP, KR) using TextMeshPro.
* **DevOps:** GitHub Actions & Git LFS for large-scale projects (~10GB+).
